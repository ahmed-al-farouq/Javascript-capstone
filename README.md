## Getting Started
  * To get a clone
    * Open your command
    * Type "git clone https://github.com/ahmed-al-farouq/Javascript-capstone.git"
  * or you can navigate to the code tab and Enjoy!

### Prerequisites
  #### knoldge of these technologies
  * HTML, CSS
  * JS & ECMA 6
  * Webpack
### Setup
  * Open your command
  * Type "git clone https://github.com/ahmed-al-farouq/Javascript-capstone.git"
  * npm i
  * npm run start
### Install
  * Open command
  * npm i
### Run tests
  * npm run test

### Deployment
  * npm run build
